## 🚀 Tasks Overview

### 1. BMI Calculator
- **Description:** A tool that calculates Body Mass Index (BMI) based on user input for weight and height.
- **Features:** Categorizes the result into Underweight, Normal, Overweight, or Obese.
- **Concepts:** Input handling, mathematical logic, and conditional statements.

### 2. Random Password Generator
- **Description:** A security utility to generate strong, randomized passwords.
- **Features:** Uses a combination of uppercase letters, lowercase letters, digits, and special characters.
- **Concepts:** Python `random` and `string` libraries.

### 3. Basic Voice Assistant
- **Description:** A CLI-based assistant to help with basic daily tasks.
- **Features:** - Tells the current system time.
  - Automates Google searches directly from the terminal.
- **Concepts:** `webbrowser` and `datetime` modules.
